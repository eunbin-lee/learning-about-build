## Module의 종류

1. Built-in Core Module (ex. Node.js module) <br>
   실행 환경에서 바로 사용 가능한 모듈

2. Community-based Module (ex. NPM) <br>
   외부에서 가져와서 사용하는 모듈 (ex. `npm install <ModuleName>`)

3. Local Module <br>
   특정 프로젝트에 정의된 모듈

<br>
<br>

## Module을 사용한다면

1. 코드의 재상용성이 증가한다
2. 코드의 관리가 편해진다
3. 코드를 모듈화하는 기준이 명확해야 한다
